# Thaumcraft Legacy Rebuild 1.19.2 — v11.63.31

## Стадия

**Simple Consumables and Legacy Jar Label** — материализация четырёх оставшихся простых ID из общего `TC4ResearchComponentItem`.

## Реализовано

### Zombie Brain / Мозг зомби

- 4 единицы голода и коэффициент насыщения `0.2`;
- считается мясной пищей;
- 80% шанс Hunger I на 30 секунд;
- после употребления 10% шанс получить 1 sticky warp;
- иначе выдаётся 1–3 permanent warp;
- изменение искажения выполняется только сервером.

### Triple Meat Treat / Тройное мясное лакомство

- 6 единиц голода и насыщение `0.8`;
- можно есть при полном голоде;
- считается мясной пищей;
- 66% шанс Regeneration I на 5 секунд.

### Knowledge Fragment / Фрагмент знаний

- правый клик расходует один предмет;
- каждый из шести первичных аспектов независимо получает 1–2 очка;
- после изменения выполняется `ThaumcraftNetwork.syncAspectKnowledge`.

### Legacy Jar Label / Бирка для банки

- `tc4_label` теперь создаётся как настоящий `JarLabelItem`;
- сохраняется NBT-аспект `Aspect`;
- работают банки, фильтрованные банки, трубы и перегонные кубы;
- динамический рецепт принимает и `jar_label`, и `tc4_label`;
- результат рецепта остаётся каноническим `jar_label`, чтобы не создавать новые дубли.
