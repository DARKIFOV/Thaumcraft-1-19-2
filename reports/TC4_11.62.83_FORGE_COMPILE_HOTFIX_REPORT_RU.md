# Thaumcraft Legacy Rebuild 11.62.83 — Forge compile hotfix

**Дата:** 2026-07-15  
**Платформа:** Minecraft 1.19.2, Forge 43.5.2, Java 17  
**Источник диагностики:** GitHub Actions run `79587666588`  
**Объективный статус:** `STATIC PASS / BUILD REQUIRES CI RERUN / RUNTIME NOT TESTED`

## 1. Результат разбора CI

GitHub Actions успешно скачал Forge-зависимости и дошёл до настоящей задачи `:compileJava`. Компилятор остановился после лимита в 100 ошибок и вывел 100 предупреждений. Большинство ошибок было каскадом от двух независимых корневых причин.

### 1.1. Traveling Trunk

В `TravelingTrunkEntity` был импортирован несуществующий для Mojmap 1.19.2 класс:

```java
net.minecraft.world.entity.animal.TamableAnimal
```

Исправленный импорт:

```java
net.minecraft.world.entity.TamableAnimal
```

Из-за неверного базового класса компилятор не воспринимал Traveling Trunk как Entity/Mob/TamableAnimal. Поэтому далее появлялись десятки вторичных ошибок в целях AI, EntityType, item-spawner, renderer, capability, NBT и override-методах.

### 1.2. Taint Spore

`TaintSporeEntity` вызывал контракт проверки заражённой колонны, который отсутствовал в исходном дереве, использованном CI. В 11.62.83 добавлен стабильный публичный метод:

```java
TaintSpreadRuntime.isColumnTainted(ServerLevel, BlockPos)
```

`TaintSporeEntity` переведён на новый метод. Старое имя `isTaintedColumn` сохранено как wrapper, чтобы не ломать существующую логику распространения и старые patches.
