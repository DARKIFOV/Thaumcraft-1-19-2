## 2. Три оставшихся compile-блокера

### 2.1. Traveling Trunk: видимость override

`Animal#mobInteract(Player, InteractionHand)` на целевой поверхности является `public`. В `TravelingTrunkEntity` override оставался `protected`, поэтому Java отклонила более слабый уровень доступа.

Исправление:

```java
@Override
public InteractionResult mobInteract(Player player, InteractionHand hand) {
```

### 2.2. Taint Spore: поздний BlockPos API

`BlockPos.containing(double, double, double)` отсутствует в Mojmap 1.19.2, использованном Forge CI.

Исправление сохраняет floor-семантику без позднего API:

```java
BlockPos stem = new BlockPos(
        Mth.floor(getX()),
        Mth.floor(getBoundingBox().minY - 0.05D),
        Mth.floor(getZ())
).below();
```

### 2.3. Taint spread: no-arg canBeReplaced

Три вызова `BlockState#canBeReplaced()` без аргументов не существуют в целевой ветке. Компилятор видит только overload с `BlockPlaceContext` и overload с `Fluid`.

Для существующей state-only логики заменяемости применён доступный в 1.19.2 материал:

```java
state.getMaterial().isReplaceable()
```

Заменены все три места: `spreadFibres`, `isTaintable`, `canFallBelow`.
