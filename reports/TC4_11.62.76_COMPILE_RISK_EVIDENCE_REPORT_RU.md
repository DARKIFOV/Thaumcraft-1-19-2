# Thaumcraft Legacy Rebuild 11.62.76 — compile-risk hardening и протокол доказательств

## 1. Исправления compile/runtime-risk

### Traveling Trunk

- заменён несовместимый/сомнительный вызов `onGround()` на Mojmap-путь `isOnGround()`;
- добавлен Forge `ForgeCapabilities.ITEM_HANDLER` через `InvWrapper`;
- capability инвалидируется вместе с сущностью;
- очистка инвентаря сохраняет размер 27 слотов;
- загрузка NBT восстанавливает состояние крышки напрямую, без звука открытия во время чтения мира;
- item-handler доступен автоматизации порта и Forge-механизмам, но runtime-поведение ещё не подтверждено.

### Outer Lands live population

- адаптер не обрабатывает незагруженный чанк и не должен инициировать его загрузку;
- отметка `populatedChunks` записывается только после возврата из генератора;
- повторная обработка проверяется через SavedData до запуска генерации;
- это не закрывает P0 Outer Lands: портал, полный лабиринт, комнаты, лут, Warden, прогрессия и возврат не проверены.

## 2. Строгий runtime manifest

Добавлен `runtime_artifacts/runtime_test_manifest.template.json` с 47 тестами. Для результатов `PASS`, `PARTIAL` и `FAIL` обязательны:

- существующий файл артефакта;
- тип артефакта;
- SHA-256;
- совпадение вычисленного SHA-256 с manifest.

`NOT_TESTED` и `N/A` не могут содержать артефакты. Отдельный негативный self-test подтвердил, что `PASS` без файла отклоняется.

## 3. Визуальное сравнение

`tools/compare_visual_artifacts.py` создаёт:

- усиленный absolute-difference PNG;
- MAE и RMSE по RGB;
- долю полностью совпавших RGBA-пикселей;
- глобальный SSIM яркости;
- SHA-256 исходных изображений.

Метрики являются диагностикой. Автоматический visual PASS запрещён; необходим side-by-side просмотр оригинала TC4 и порта.

## 4. CI

Оба workflow теперь:

1. выполняют source/resource guards;
2. валидируют template runtime manifest;
3. запускают `gradlew build`;
4. сохраняют `build/ci/gradle-build.log` даже при падении;
5. после загрузки лога завершают job с ошибкой;
6. только после успешной сборки вычисляют SHA-256 JAR и обновляют объективный отчёт.
