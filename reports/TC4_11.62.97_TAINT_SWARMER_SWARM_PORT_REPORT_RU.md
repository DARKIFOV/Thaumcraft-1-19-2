# Thaumcraft Legacy Rebuild 11.62.97 — порт Spore Swarmer / Taint Swarm

**Цель:** продолжение переноса Thaumcraft 4.2.3.5 с Minecraft 1.7.10 на Forge 1.19.2.

## Реализовано

- Добавлен `TaintSporeSwarmerEntity` с оригинальными базовыми параметрами: размер 1×1, 75 здоровья, 500-тиковый счётчик, выпуск одного роя рядом с игроком и выпуск роя при смерти.
- Восстановлена ветка `BlockTaint`: шанс 1/200 для заражённой корки, проверка отсутствия другого роевика в радиусе 16 блоков, удаление корки и появление сущности.
- Добавлен `TaintSwarmEntity`: невидимое тело, облако частиц, полёт по заражённым колонкам, поиск игрока в радиусе 12 блоков, атака с ядом на 100 тиков, сохранение скорости жертвы, режим summoned и саморазрушение.
- Добавлены оригинальные звуковые события `roots`, `gore`, `swarmattack` и выпадение двух случайных материалов: tainted slime / taint tendril.
- Перенесена двухкубовая модель `ModelTaintSporeSwarmer`, исходная текстура `textures/models/taint_spore.png`, пульсация и full-bright ядро.
- Зарегистрированы оба `EntityType`, атрибуты, клиентские renderer/model layer и локализация EN/RU.
- Версия проекта, manifest и CI/release workflow подняты до `11.62.97`.

## Ограничение сборки

Локальный `./gradlew build` не дошёл до компиляции: Gradle Wrapper не смог разрешить `services.gradle.org` (`UnknownHostException`). В контейнере также доступна Java 21, тогда как проект и официальный Forge 1.19.2 workflow используют Java 17. Поэтому JAR не заявлен как собранный, а поведение в клиенте/сервере остаётся `NOT TESTED`.

## Основание переноса

Поведение и модель сверены с исходниками TC4 1.7.10 (`EntityTaintSporeSwarmer`, `EntityTaintSwarm`, `ModelTaintSporeSwarmer`, `RenderTaintSwarm`, `BlockTaint`). Регистрация сохранена через существующий `DeferredRegister` проекта, а клиентские renderer/layer регистрации размещены на mod event bus в принятом для Forge 1.19.2 пути.
