# Thaumcraft Legacy Rebuild 11.62.96 — P0 item geometry and UV hotfix

**Дата:** 2026-07-16  
**Платформа:** Minecraft 1.19.2, Forge 43.5.2, Java 17  
**Основа:** 11.62.95 All-Block Render Layer & Geometry Hotfix  
**Статус:** `STATIC PASS / COMPILE REQUIRES CI / RUNTIME VISUAL NOT TESTED`

## 1. Область исправления

Версия 11.62.95 восстановила world-геометрию и RenderType ряда блоков, но несколько постоянно видимых P0-предметов всё ещё использовали плоские иконки или упрощённые item-модели. В 11.62.96 исправлены шесть канонических item paths:

1. Bellows;
2. Alchemical Centrifuge;
3. Infusion Matrix;
4. Brain in a Jar;
5. Essentia Reservoir;
6. Advanced Alchemical Furnace.

Миграционные alias IDs сохранены. Там, где alias не имеет собственного `BlockItem`/BEWLR, оставлена безопасная статическая иконка, чтобы старые стеки не стали невидимыми.

## 2. Геометрия и UV

### Bellows

Добавлена отдельная BEWLR-модель из пяти частей оригинального `ModelBellows`: три доски, сопло и кожаный мешок. Для мешка сохранена отдельная нормализация UV 64×32, тогда как каркас использует 128×64. Это существенно: одно и то же числовое смещение UV даёт другую область при неверном размере атласа.

### Alchemical Centrifuge

Добавлена точная шестичастная модель `ModelCentrifuge`: crossbar, два dingus, core, top и bottom. Item renderer рисует все шесть частей; world renderer использует те же UV и вращает только внутреннюю сборку.

### Infusion Matrix

Item path переведён на BEWLR и использует существующий `TC4InfusionMatrixModel`. Восемь элементов матрицы рисуются одной и той же исходной UV-развёрткой с независимыми поворотами.

### Brain in a Jar

Предмет больше не является плоской иконкой. Renderer объединяет реальную block-модель банки, оригинальную модель мозга и отдельный полупрозрачный объём brine.

### Essentia Reservoir

Оригинальный `reservoir.obj` преобразован из Z-up в Y-up без изменения топологии, списка UV и индексов граней. Одна OBJ-модель теперь используется и блоком, и предметом.

### Advanced Alchemical Furnace

Для item-контекста создана центрированная и равномерно нормализованная копия оригинального `adv_alch_furnace.obj`. Изменены только координаты вершин; UV и face indices сохранены. Добавлены отдельные display transforms для GUI, ground, fixed, first-person и third-person.

## 4. Что не подтверждено

Локальный Gradle wrapper не разрешил `services.gradle.org`, поэтому `compileJava` и JAR для 11.62.96 не получены. Статические проверки не доказывают фактический масштаб, освещение, translucent sorting, culling и положение в руках. Для runtime PASS требуются клиентские снимки каждого из шести предметов в GUI, ground, fixed, first-person left/right и third-person left/right.

Отдельным последующим этапом остаются:

- динамическое переключение active-текстур Advanced Alchemical Furnace;
- анимация сжатия Bellows в item/world runtime;
- визуальная настройка скорости вращения Centrifuge;
- проверка прозрачности brine Brain Jar.
