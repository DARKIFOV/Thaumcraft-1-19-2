# Thaumcraft Legacy Rebuild 11.62.98

## Перенос Taint Poison и заражённых ресурсов

**Цель:** продолжить перенос Thaumcraft 4.2.3.5 с Minecraft 1.7.10 на Minecraft 1.19.2 / Forge 43.5.2.

**База:** 11.62.97 — Taint Spore Swarmer / Taint Swarm.

## Что перенесено

### Отдельный эффект Taint Poison

Добавлен зарегистрированный `thaumcraft:taint_poison`, повторяющий контракт `PotionFluxTaint`:

- категория `HARMFUL`;
- исходный цвет `0x663377`;
- срабатывание каждые `40 >> amplifier` тиков;
- лечение сущностей с маркером `TaintedMob` на 1 здоровье;
- 1 единица магического урона остальным живым не-нежитям;
- исходная иконка из ячейки `(3,1)` TC4-атласа `textures/misc/potions.png` экспортирована в современный путь `textures/mob_effect/taint_poison.png`;
- локализации: `Taint Poison` / `Магический яд`.

Маркер заражённого существа подключён к паучку, споре и её роевику, рою, обычному/гигантскому щупальцу и Thaumic Slime. Малые и производные сущности наследуют маркер от базового класса.

### Источники эффекта

Обычный `MobEffects.POISON` заменён на выделенный эффект в подтверждённых цепочках TC4:

- Taint Block;
- Taint Fibres;
- атака Taint Swarm;
- событие нестабильности Infusion Matrix.

Очищающие механизмы порта теперь снимают оба эффекта — обычный яд и `Taint Poison`: Fume Dissipator, Cleansing Talisman, TT Fume Dissipator и очищающий импульс Pure Aura Node.

### Заражённая слизь и щупальце

Добавлен `TaintedResourceItem`, повторяющий `ItemResource` metadata 11/12:

- серверная проверка в инвентаре живой сущности;
- нежить не заражается;
- повторное наложение не выполняется, пока эффект активен;
- исходная вероятность `nextInt(4321) <= stack size`;
- длительность 120 тиков;
- игрок получает локализованное предупреждение;
- у игрока расходуется один предмет, кроме creative mode.

Функциональный класс подключён к активной заражённой слизи/`Tainted Goo` и миграционному предмету `tc4_taint_tendril`.

### Дроп Taint Spider

Восстановлена исходная ветка:

- шанс дропа 1 из 6;
- затем равный выбор между Tainted Goo и Taint Tendril.

## Сборка

Команда:

```text
./gradlew compileJava --no-daemon --stacktrace
```

не дошла до `compileJava`: Gradle Wrapper не смог разрешить `services.gradle.org` при загрузке `gradle-7.5.1-bin.zip` (`UnknownHostException`). В контейнере доступна Java 21, тогда как проект и Forge 1.19.x рассчитаны на Java 17.

Следовательно:

- JAR не получен;
- реальная компиляция Mojmap/Forge не подтверждена;
- клиентская графика, cadence эффекта, статистика выпадения, multiplayer sync и dedicated server остаются `NOT_TESTED`.

Лог: `reports/TC4_11.62.98_LOCAL_GRADLE_BUILD_ATTEMPT.log`.
