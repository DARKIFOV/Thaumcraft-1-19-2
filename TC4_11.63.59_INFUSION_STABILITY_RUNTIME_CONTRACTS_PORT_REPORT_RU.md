# TC4 v11.63.59 — Original Infusion Stability Runtime Contracts

## Цель

Продолжить перенос оригинального Thaumcraft 4.2.3.5 после v11.63.58 и устранить доказуемые расхождения в расчёте симметрии и допустимых стабилизаторов Infusion Matrix.

## Исправления runtime

### Единственный проход расчёта нестабильности

Оригинальная матрица начинает обычную и enchantment-инфузию со значением `symmetry + recipeInstability`. В порте симметрия уже включала пары/непарные стабилизаторы, но auxiliary-отчёт применял их второй раз. Теперь стартовая формула выполняется один раз и ограничивается диапазоном 0–25.

Во время процесса нестабильность не уменьшается автоматически: пересканирование окружения может только сохранить прежнее значение либо поднять его, если симметрия ухудшилась.

### Точный whitelist стабилизаторов

Стабилизаторами снова считаются только:

- блоки, реализующие `InfusionStabilizer`;
- допустимые обычные головы/черепа;
- Tallow Candle через тот же marker-контракт;
- семь crystal cluster blocks, теперь зарегистрированных как `CrystalClusterBlock implements InfusionStabilizer`.

Исключены ошибочные structural-объекты: infusion pillar, arcane stone bricks, node stabilizer и dragon head.
