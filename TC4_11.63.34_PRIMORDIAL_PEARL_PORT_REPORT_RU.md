# Thaumcraft Legacy Rebuild v11.63.34 — Изначальная жемчужина

## Эталон

Стадия сверена с `ItemEldritchObject` metadata 3 из Thaumcraft 4.2.3.5.

## Перенесено

- `tc4_eldritch_object_3` больше не создаётся общим fallback-классом.
- Совместимый ID `primordial_pearl` использует тот же функциональный класс и больше не является неработающим клоном.
- Восстановлены размер стака 1, эпическая редкость, исходная текстура и две строки tooltip.
- ПКМ жемчужиной по узлу ауры расходует предмет и изменяет базовые аспекты узла.
- Составные аспекты могут потерять единицу, первичные получают исходный случайный диапазон.
- Узел может улучшиться `FADING -> PALE -> NORMAL -> BRIGHT` по исходным шансам.
- Исследование `PRIMNODE` использует улучшенные диапазоны оригинала.
- Восстановлены взрыв и 33 попытки размещения Flux Goo/Flux Gas вокруг узла.
- Для legacy-ID добавлен исходный набор аспектов 16 каждого первичного аспекта.

## Исправление входного архива

Во входном v11.63.33 отсутствовал весь пакет `com.darkifov.thaumcraft.world`, хотя отчёт
версии заявлял его восстановленным. Шесть файлов Magical Forest/worldgen восстановлены
из локально собранной рабочей базы v11.63.24.

## Runtime

Требуется проверить расход жемчужины, обе ветки PRIMNODE, изменение всех типов узлов,
силу взрыва, распространение flux и синхронизацию между двумя клиентами.
