# Thaumcraft Legacy Rebuild v11.63.01 — отчёт портирования

База: `v11.63.00_LIQUID_DEATH_DAMAGE_SOURCE_SOURCE`, Forge 1.19.2 / 43.5.2, Java 17.

## Исправлено

- Полный каталог из 216 крафтящихся NBT-вариантов: 18 стержней × 6
  набалдашников × обычная/скипетр-форма. Варианты добавляются в creative и
  становятся отдельными JEI subtype по `rod:cap:sceptre`.
- Vis больше не определяется registry-ID предмета. Крафтовый результат получает
  оригинальные корневые теги TC4 `rod`, `cap`, `sceptre`; ёмкость вычисляется из
  стержня и равна `rodCapacity * 100` centivis, для скипетра — `* 150`.
- Creative-варианты заполняются до собственной ёмкости, а vis/focus не дробят
  один и тот же компонентный subtype в JEI.
- Рендер жезла переведён с приблизительной cuboid-UV на точный порядок вершин и
  UV-прямоугольников `ModelBox` Minecraft 1.7.10 с `mirror=true`.
- Каждая грань стержня/набалдашника теперь передаёт корректную преобразованную
  нормаль вместо общей `(0,0,1)`, что исправляет плоское/неправильное освещение.
- Активные PNG всех стержней и набалдашников сверены с банком оригинальных
  текстур TC4 и совпадают побайтно.
- Исправлен неверный импорт `Heightmap` в `TaintSwarmEntity`, блокировавший
  компиляцию исходников v11.63.00.
- Отключена параллельная сборка ForgeGradle 5.1: на Windows она создавала гонку
  mapped-JAR после `clean` и запускала javac без Minecraft classpath.
- GitHub Actions, release bundle и runtime manifest обновлены на v11.63.01.
