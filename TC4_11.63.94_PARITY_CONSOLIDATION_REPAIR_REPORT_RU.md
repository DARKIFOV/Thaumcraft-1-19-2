# Thaumcraft Legacy Rebuild v11.63.94 — восстановление и консолидация parity

## Исходная ситуация

Для проверки использованы три архива:

- пользовательский `TC4_v11.63.93_WORLDGEN_ESSENTIA_V2.zip`;
- чистый восстановленный `v11.63.72`;
- исходная база `v11.63.70`.

Архив с именем `v11.63.93` не являлся пригодной релизной базой. Его manifest называл версию 11.63.93, но `build.gradle` и `mods.toml` оставались на 11.63.92. Десять parity-классов были синтаксически повреждены автоматическими вставками: методы дублировались, оказывались внутри других методов или после закрывающей скобки класса. В GameTest-файле присутствовала двойная аннотация перед одним методом и отсутствовала аннотация у следующего. Поэтому заявленные 109 GameTest не означали 109 компилируемых тестов.

## Выбранная база

За основу принят чистый `v11.63.72`, поскольку он сохраняет source-linked константы Wand и Essentia. Из `v11.63.93` перенесены только механики, которые удалось независимо сверить с production-кодом и оригинальным исходником TC4. Фиктивные контракты, проверявшие только количество Java-файлов, исключены.

## Добавленные и исправленные контракты

- **Taint:** шесть специальных замен при смерти, slime fallback, смещения `nextInt(3)-1 / nextInt(5)-3 / nextInt(3)-1`, шанс споры `1/200`.
- **Worldgen:** вес Magical Forest `5`, два биома, точки входа greatwood/silverwood, десять попыток mana pod.
- **Runic:** shield `2000 ms / 80 ticks / 50 vis`, флаги taint/dissolve damage sources, три части fortress armor и делители `25/35/20`.
- **Eldritch:** строгие пороги Warp `>10/>25/>50`, Crimson Rites `+1 sticky warp / +5 attunement`, константы Outer Lands.
- **Aura:** шесть типов узлов и цвета, четыре модификатора, 75% ухудшение при помещении в jar, сохранение аспектов 100%.
- **Golem v2:** девять декораций, шесть bell-режимов, формула переносимого количества при Earth-upgrade.
- **Wand focus:** десять фокусов и 21 непрерывный upgrade ID `0..20`.
- **Alchemy:** production-код использует общий helper точной формулы Liquid Death `max(1, (1 + nextInt(amount)) / 2)`.
- **Recipes:** 48 aspect labels, denominator оригинальных recipe registrations `258`, отдельные 18 smelting-bonus registrations.
