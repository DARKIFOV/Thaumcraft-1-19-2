# Thaumcraft Legacy Rebuild v11.63.48 — OreDictionary, наконечники жезлов и старые печные рецепты

**Дата:** 2026-07-19  
**Цель:** Minecraft 1.19.2, Forge 43.5.2, Java 17  
**Эталон:** Thaumcraft 4.2.3.5

## Результат этапа

Из исторического recipe-ledger дополнительно закрыты **14 точных source-записей**. Совокупный доказанный runtime-путь теперь имеют **213 из 243** оригинальных записей; неразрешённый остаток уменьшен до **30**.

### Legacy OreDictionary → Forge tags

Три crucible-рецепта используют прямые современные аналоги старых OreDictionary-имён:

- `oreTin` → `#forge:ores/tin`;
- `oreSilver` → `#forge:ores/silver`;
- `oreLead` → `#forge:ores/lead`.

Результаты и стоимость сохранены: один соответствующий рудный кластер, `METALLUM 1 + ORDO 1`.

Для arcane-рецептов наконечников заменены внутренние жёсткие ID:

- `nuggetCopper` → `#forge:nuggets/copper`;
- `nuggetSilver` → `#forge:nuggets/silver`;
- `nuggetThaumium` → `#forge:nuggets/thaumium`;
- `nuggetVoid` → `#forge:nuggets/void_metal`.

Золотой наконечник по-прежнему требует точный `minecraft:gold_nugget`, как в исходнике.

`ArcaneWorkbenchRecipe` теперь разбирает и сопоставляет tag-катализатор тем же механизмом, который уже применялся к обычным ингредиентам. Это устраняет различие между JSON, runtime и JEI-представлением рецепта.

## Полное семейство наконечников

Закреплены пять arcane-рецептов:

- Gold: стоимость 3 Aer/Ignis/Ordo;
- Copper: стоимость 2 Aer/Ignis/Ordo;
- Silver Inert: стоимость 4 Aer/Ignis/Ordo;
- Thaumium Inert: стоимость 6 Aer/Ignis/Ordo;
- Void Inert: `Perditio 27`, `Ordo 27`, `Ignis 18`, `Aer 18`.

Также закреплены три infusion-рецепта заряженных наконечников:

- Silver: 2 Salis Mundus, instability 4, `Potentia 8 + Auram 4`;
- Thaumium: 3 Salis Mundus, instability 5, `Potentia 12 + Auram 6`;
- Void: 4 Salis Mundus, instability 8, по 18 `Potentia/Vacuos/Alienis/Auram`.

## Старые печные рецепты

В точный ledger переведены:

- Greatwood/Silverwood Log → Charcoal, `0.5 XP`, 200 тиков;
- Cinnabar Ore → Quicksilver Drop, `1.0 XP`, 200 тиков;
- Amber Ore → Amber, `1.0 XP`, 200 тиков.

## Автоматические проверки

Добавлены два обязательных server-only Forge GameTest:

1. pure-metal crucible tags и три legacy-smelting рецепта через настоящий `RecipeManager`;
2. пять arcane и три infusion рецепта наконечников с проверкой входов, Vis, компонентов, instability и essentia.

Runtime-манифест расширен шестью сценариями до **336**, обязательный GameTest-пакет — до **14 тестов**.

## Ограничения доказательств

Статический guard подтверждает исходный код, ресурсы и формулы, но не доказывает, что сторонний мод действительно заполняет каждый `forge:*` tag подходящими предметами. Для этого требуется Java 17 Forge-сборка и игровой запуск с совместимым content-модом.

`runGameTestServer` в текущем окружении не запускался. Все новые runtime-сценарии остаются `NOT_TESTED`.
