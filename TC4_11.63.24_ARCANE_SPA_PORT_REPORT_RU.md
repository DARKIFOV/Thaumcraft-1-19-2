# Thaumcraft Legacy Rebuild 1.19.2 — v11.63.24

## Стадия

**Arcane Spa / Магический спа, Bath Salts и Purifying Fluid** — восстановление оставшихся контрактов `TileSpa` из Thaumcraft 4.2.3.5 поверх уже существующей функциональной базы Forge 1.19.2.

## Реализовано

- 5000 mB и расход ровно 1000 mB на успешный цикл.
- Один слот только для Bath Salts; автоматизация и жидкость недоступны с верхней стороны.
- Режим смешивания создаёт Purifying Fluid только из ванильной воды и одной порции соли.
- Режим без смешивания размещает хранимую world-placeable жидкость.
- Восстановлен независимый для каждого block entity счётчик `counter++ % 40`, вместо глобальной синхронизации по world time.
- Красный камень блокирует цикл выдачи.
- После первого source над устройством поиск идёт по горизонтальному слою 5×5.
- Источник-сосед проверяется по всем шести направлениям, как старый `BlockUtils.isBlockTouching`.
- Позиция должна быть заменяемой, иметь прочную опору снизу и не содержать block entity.
- Ресурсы списываются только после успешного `setBlock`.
- Bath Salts живут 200 тиков и превращают source-water в Purifying Fluid.
- Source Purifying Fluid выдаёт Warp Ward по сохранённой TC4-формуле и исчезает.
- GUI сохраняет исходные координаты бака/переключателя и звук `thaumcraft:cameraclack`.
