# Thaumcraft Legacy Rebuild v11.64.33 — полное закрытие системы wand foci

## Единственная цель раунда

Закрывалась одна цельная система: десять оригинальных wand foci TC4, установка/снятие фокуса, Focus Pouch, upgrade NBT, vis-cost/cooldown contracts, HUD/render и серверная смена через `WandManager.changeFocus`.

## Оригинальные пути

Проверены `ItemWandCasting`, `WandManager`, `ItemFocusBasic`, `ItemFocusPouch`, `ItemFocusPouchBauble`, `InventoryFocusPouch`, `ContainerFocusPouch`, `GuiFocusPouch`, `KeyHandler`, `PacketFocusChangeToServer`, `REHWandHandler` и классы Fire/Frost/Shock/Excavation/Portable Hole/Equal Trade/Warding/Nine Hells/Pech's Curse/Primal.

## Исправленные расхождения

- Удалено двойное хранение `Focus` + `focus`. Канон — только compound `focus` с полным ItemStack.
- Старый upper-case `Focus` оставлен только как одноразовая миграция и после чтения удаляется.
- Upgrade NBT полного focus stack сохраняется при установке, снятии, pouch swap и перезапуске.
- Удалён выдуманный `SelectedFocus` из Focus Pouch.
- Focus Pouch снова сохраняет только `Inventory` list с byte `Slot`, имеет 18 слотов и открывает исходный GUI.
- Удалены выдуманные tooltip-инструкции и прямой cycle/equip через right-click.
- Right-click wand больше не ставит focus из off-hand, не листает pouch и не снимает focus при приседании.
- F снова является отдельным штатным путём: удержание открывает radial; Shift+F отправляет `REMOVE`.
- Добавлен original-style radial selector на `radial.png`/`radial2.png`, с радиусом `16 + count*2.5`, равными секторами и TreeMap sorting helper.
- Packet применяется только к non-sceptre wand в main hand; off-hand wand больше не принимается.
- Client radial и server WandManager используют один и тот же список/sort key.
- Transactional swap возвращает новый focus в точный исходный слот при невозможности сохранить старый.
- Название Focus Pouch GUI переведено на `container.focuspouch`.
