# GitHub upload — v11.62.41

## Загрузка проекта

1. Открой репозиторий мода на GitHub.
2. Загрузи содержимое корня архива, а не сам внешний ZIP.
3. В корне репозитория должны находиться `build.gradle`, `settings.gradle`, `gradlew`, папки `.github`, `gradle`, `src` и `tools`.
4. Удали старые исходники перед заменой либо загрузи эту версию в отдельную ветку.
5. Сделай commit и push в `main` или `master`.

Workflow **Build and audit** автоматически:

- использует Java 17;
- проверяет, что проект остаётся Forge 1.19.2;
- запускает проверки GUI, worldgen, Таумометра, листвы, жезлов и Исследовательского стола;
- проводит аудит реестра и ресурсов;
- собирает Forge JAR.

## Получение JAR

1. Открой вкладку **Actions**.
2. Открой последний успешный запуск **Build and audit**.
3. Скачай artifact `thaumcraft-legacy-rebuild-v11.62.41`.
4. Распакуй artifact.
5. Помести в `mods` только обычный JAR, не `sources` JAR.
6. Удали из `mods` все предыдущие JAR этого мода.

## GitHub Release

После зелёной сборки создай тег:

```bash
git tag v11.62.41
git push origin v11.62.41
```

Workflow **Publish GitHub release** повторно выполнит проверки, соберёт JAR и приложит release ZIP и SHA-256.

## Обязательный тест

Используй Minecraft 1.19.2, Forge 43.5.2 и Java 17. Старый мир предварительно скопируй.

Проверь:

- Greatwood и Silverwood в новом мире, Fast/Fancy graphics;
- отсутствие внутренних синих граней в плотной Серебряной кроне;
- опадание листьев после удаления ствола;
- полный жезл с наконечниками в GUI, первой и третьей руке;
- обычный жезл, посох и скипетр с несколькими фокусами;
- создание Исследовательского стола из двух столов во всех четырёх направлениях;
- видимость чернильницы, пера, пергамента и исследовательской записки;
- открытие стола с обеих визуальных сторон;
- Таумометр на блоках, сундуках, существах и выброшенных предметах;
- `/locate biome thaumcraft:magical_forest` в новом мире;
- банки, Рунную матрицу, Очки откровения, стабилизатор и преобразователь узла.
