# Thaumcraft Legacy Rebuild 11.63.02 — перенос Магического леса

## Эталон TC4 4.2.3.5

Источник сравнения: `BiomeGenMagicalForest.java`, `Config.java` и
`ThaumcraftWorldGenerator.initialize()` оригинального мода для Minecraft 1.7.10.

- вес биома по умолчанию: 5;
- регистрация: одновременно WARM и COOL;
- температура/осадки: 0.7 / 0.6;
- цвет травы: `0x55FF81`;
- цвет листвы: `0x66FFC5`;
- цвет воды: `0x0077EE` (30702);
- дополнительные спавны: волк 2/1–3, лошадь 2/1–3, ведьма 3/1,
  эндермен 3/1, печ 10/1–2, висп 10/1–2.

- Через Forge biome modifier восстановлены оригинальные дополнительные спавны.
- Сохранены сериализуемый источник биомов и видимость для `/locate biome`.
