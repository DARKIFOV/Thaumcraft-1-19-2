Это **авторитетный источник истины** для точного портирования. Пользователь
предоставил полный оригинальный исходник TC4 4.2.3.5.

## Файлы

- `Thaumcraft4-1.7.10-4.2.3.5-source.zip` — полный исходный код оригинала
  (859 `.java`: `thaumcraft/common` 521, `thaumcraft/client` 232, `thaumcraft/api` 61,
  `thaumcraft/codechicken` 41), плюс `assets/thaumcraft/lang/*`, шейдеры и ресурсы.

## Как пользоваться (в новом чате)

```sh
cd /data/port/reference
mkdir -p _oracle && cd _oracle
unzip -q ../Thaumcraft4-1.7.10-4.2.3.5-source.zip
# оригинал: _oracle/Thaumcraft4-1.7.10-master/thaumcraft/...
```

Например, эталон warp-рантайма:
`_oracle/Thaumcraft4-1.7.10-master/thaumcraft/common/lib/WarpEvents.java`

## Правила использования оракула

1. Каждое расхождение (parity-класс) должно ссылаться на конкретный файл/строку
   оригинала из этого архива. НЕ выдумывать расхождения.
2. Оригинал — это 1.7.10 (Forge/FML, obf-имена `func_*`/`field_*`). Портируем
   ПОВЕДЕНИЕ, а не сигнатуры: маппинг 1.7.10 -> 1.19.2 держим в `tc4_source_mapping`.
3. Числовые пороги, RNG-распределения, тексты и порядок операций копируем 1:1.
4. `reference/` лежит ВНЕ `src/main/java`, поэтому Gradle его не компилирует.
   Не переносить оригинальные `.java` в `src/` — они не соберутся под 1.19.2.
