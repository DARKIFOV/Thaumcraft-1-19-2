# Thaumcraft Legacy Rebuild 1.19.2 — v11.63.33

## Стадия

Простая связанная семья: `tc4_essence`, три мешка с добычей, `tc4_crimson_rites`, `tc4_eldritch_object` и `tc4_eldritch_object_2`.

## Реализовано

- `TC4EssenceItem extends EssentiaPhialItem`: пустой/наполненный NBT-контракт, 8 единиц эссенции, творческие варианты для всех аспектов, совместимость с банками, алембиками, тигелем, трубами и големами.
- Двухслойная модель исходного пузырька: byte-exact `phial.png` и анимированный `essence.png`, окрашиваемый цветом аспекта.
- Динамический рецепт бирки принимает любой `EssentiaPhialItem` и возвращает пустой предмет того же registry ID.
- `TC4LootBagItem`: лимит стака 16, common/uncommon/rare, 8–12 вызовов генератора добычи и исходный звук монет.
- `TC4CrimsonRitesItem`: серверное изучение CRIMSON без расхода книги.
- `TC4EldritchObjectItem`: отдельные варианты Eldritch Eye и Runed Tablet, исходная редкость и tooltip.
- Eldritch Altar принимает точный legacy Eye; Eldritch Lock принимает точный Runed Tablet.
- Удалён дублирующий общий event-handler: поведение принадлежит конкретным item-классам.
- Исправлено повторное объявление `int slot` в `TC4OuterLandsLootAdapter`, являвшееся реальной Java compile-error.
