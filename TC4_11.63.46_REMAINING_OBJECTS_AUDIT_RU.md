- автономный `FumeDissipatorBlockEntity`;
- Aer-питание через energized Vis relay network;
- ограниченный поиск Flux Goo/Flux Gas в радиусе 16;
- внутренние cleanup-charge и буфер Praecantatio 0–4;
- односторонний essentia endpoint;
- NBT/network sync;
- исходная модель и renderer;
- обязательный server-only GameTest.

- Java 17 `gradlew build`;
- headless `runGameTestServer` со всеми десятью обязательными тестами;
- старый мир, где legacy Flux Scrubber существовал без новой `BlockEntity`;
- несколько scrubber/amulet/workbench consumers на одном energized-node pool;
- длительная производительность радиуса 16 и chunk load/unload;
- essentia rollback в реальной разветвлённой tube-сети;
- multiplayer block-update синхронизация;
- renderer при всех шести направлениях и на разных GPU;
- fixture старых миров для отложенного мигратора.

Итог: **неперенесённых generic-ID нет**. Flux Scrubber source-parity и десятый обязательный GameTest присутствуют, но игровая готовность не считается подтверждённой без Java 17 Forge build и runtime-логов.
