# Thaumcraft Legacy Rebuild 1.19.2 — v11.63.30

## Стадия

**Passive Crafting Components / Простые компоненты крафта** — материализация ещё шести TC4-ID, которые не имеют самостоятельной игровой механики, но должны быть настоящими предметами, а не общими исследовательскими fallback-заглушками.

## Закрытые ID

- `tc4_coin` — Gold Coin / Золотая монета;
- `tc4_dust` — Salis Mundus / Соль тауматурга;
- `tc4_filter` — Vis Filter / Фильтр;
- `tc4_mirrorglass` — Mirrored Glass / Зеркальное стекло;
- `tc4_voidseed` — Void Seed / Семя Пустоты;
- `tc4_golem_upgrade_empty` — Blank Golem Upgrade / Пустое улучшение голема.

## Реализация

Первые пять предметов используют явный вид `TC4SimpleResourceItem.Kind.CRAFTING_COMPONENT`. Пустая матрица улучшения вынесена в `TC4EmptyGolemUpgradeItem`: она складывается по 64, имеет редкость `UNCOMMON` и переведённый tooltip. Все предметы сохраняют исходные registry ID, модели и byte-exact текстуры.

Добавлены теги `thaumcraft:passive_crafting_components`, `coins`, `salis_mundus`, `vis_filters`, `mirrored_glass`, `void_seeds` и `golem_upgrades/empty`. Существующие normal/arcane/infusion-рецепты продолжают ссылаться на точные `tc4_*` ID.
