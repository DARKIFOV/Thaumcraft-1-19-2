# v11.62.43 — TC4 Visual Texture Audit

## Основа

- Minecraft **1.19.2**
- Minecraft Forge **43.5.2**
- Java **17**
- эталон: предоставленные исходники Thaumcraft **4.2.3.5** для 1.7.10

## Исправлено

### Полный оригинальный банк текстур

Проведено сравнение всех ресурсов `assets/thaumcraft/textures` с оригиналом.

До исправления:

- отсутствовал `misc/wispy.png`;
- `models/scanner.png` визуально был похож, но отличался альфа-каналом;
- в каноническом каталоге было 939 файлов вместо 940.

Теперь:

- восстановлен оригинальный `wispy.png`, который использует поток vis между жезлом и узлом;
- восстановлен точный `scanner.png` вместе с оригинальной прозрачностью;
- канонический банк содержит **940 из 940** файлов;
- CI сравнивает каждый файл с SHA-1 из исходного inventory.

### Таумометр

Сверено с `ItemThaumometerRenderer` оригинального TC4:

- аспекты снова сортируются по ID/тегу, а не по количеству;
- восстановлена треугольная раскладка **5/4/3/2/1**;
- лимит увеличен с 10 до оригинальных **15 аспектов**;
- иконки возвращены к размеру 16×16;
- удалён выдуманный узел и пузырь внутри стекла прибора;
- на стекле остаются название, тип/модификатор и реальные аспекты цели;
- сам узел продолжает отображаться отдельным мировым рендерером, как в TC4.

### Узел ауры

Сверено с `TileNodeRenderer`:

- удалено отсутствующее в оригинале увеличение узла на 10% после energize;
- удалён дополнительный цветной sprite-слой после высасывания vis;
- сохранён оригинальный отдельный поток `wispy.png` к жезлу;
- сохранено разделение additive и alpha blend для типов и аспектов.

### Жезлы, посохи и скипетры

Сверено с `ModelWand`:

- скипетр снова имеет независимое кольцо из **10 вращающихся рун**, даже без обычного rune-upgrade;
- восстановлено вращение `36° × rune + ticks`;
- восстановлена оригинальная пульсация красного, зелёного и alpha каналов;
- руны переключены на светящийся runtime render path;
- предыдущая правка 64×32 UV-развёртки стержня и наконечников сохранена.

### Очки откровения

Причина неправильной модели была в том, что предыдущий слой растягивал произвольные части всего PNG по нескольким плоскостям. Оригинальный `goggles.png` — физический 128×64 файл, соответствующий логическому armor-атласу 64×32.

Теперь:

- используется оригинальная UV-сетка головы 64×32;
- физическое 2× разрешение PNG сохраняется;
- линзы и ремни берутся из тех же координат, которые использовала armor-модель TC4;
- случайное семплирование прозрачных областей удалено.

### Таумономикон

- браузер привязан к `textures/original/thaumcraft4/gui/gui_research.png`;
- страницы привязаны к `textures/original/thaumcraft4/gui/gui_researchbook.png`;
