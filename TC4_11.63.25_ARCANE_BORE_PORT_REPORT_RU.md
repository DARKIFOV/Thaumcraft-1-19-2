# Thaumcraft Legacy Rebuild v11.63.25 — перенос магического бура

## Эталон TC4 4.2.3.5 (Minecraft 1.7.10)

Сверка выполнена с `TileArcaneBore`, `TileArcaneBoreBase`, `ContainerArcaneBore`,
`GuiArcaneBore`, `ModelBore`, `ModelBoreEmit` и `ModelBoreBase` оригинального TC4.

## Перенесено на Forge 1.19.2

- Оба ID заменены с generic fallback на реальные блоки и BlockEntity.
- GUI 176x141 сохраняет исходные позиции слотов фокуса раскопок и кирки.
- Бур работает от красного сигнала на нём или основании; направления меняются жезлом.
- Базовая область 5x5 расширяется `ENLARGE`; глубина поиска — до 64 блоков.
- `POTENCY` объединяется с Efficiency, `TREASURE` с Fortune; поддерживается Silk Touch.
- Кирка изнашивается, добыча поступает в инвентарь у сопла либо выбрасывается наружу.
- Основание создаёт suction 128 для Perditio; единица ускоряет следующие 20 блоков.
- Инвентарь, ускорение и позиция обхода сохраняются и синхронизируются.
- Перенесены исходные cuboid-размеры и UV моделей на `Bore.png`.

## Дополнительные исправления сборки

Исправлены пропущенные renderer-imports, устаревший `setNoPhysics` и несовместимый тип
параметра урона `PrimalCrusherItem`, блокировавшие чистую сборку исходника v11.63.24.

## Проверка

- `compileJava` на Java 17: успешно.
- Runtime ещё требуется проверить в Forge: шесть направлений, сигнал, трубы Perditio,
  Fortune/Silk и вывод в сундук.
