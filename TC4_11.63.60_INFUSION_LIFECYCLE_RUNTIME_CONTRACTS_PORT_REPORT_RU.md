# TC4 11.63.60 — Infusion lifecycle runtime contracts

Закреплено исходное поведение Infusion Matrix при изменении катализатора, успешном завершении и ожидании отсутствующего компонента.

- изменённый locked catalyst запускает ровно одну weighted instability-ветку и отменяет процесс;
- отдельный generic Warp при этой отмене не начисляется;
- успешная инфузия не начисляет Warp автоматически;
- отсутствующий компонент оставляет матрицу в режиме ожидания;
- добавлено четыре обязательных server-only GameTest; общий пакет — 52 теста и 380 runtime-сценариев.

Статический source-контракт проверен. JAR и runtime PASS для этой версии не заявляются без Java 17 build и runGameTestServer.
